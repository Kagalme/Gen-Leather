💡 Quick Tips

Selamat datang di GenLeather Admin Dashboard! Gunakan menu di sidebar untuk mengelola konten website Anda:

  • Hero Section - Atur gambar, teks, dan statistik di bagian atas halaman
  • Kategori - Kelola kategori produk (Tas, Dompet, Sabuk, dll)
  • Koleksi Produk - Tambah, edit, atau hapus produk
  • Cerita / About - Edit bagian tentang kami
  • Ulasan - Kelola testimonial pelanggan
  • Footer - Atur informasi kontak dan sosial media

💾 Semua perubahan disimpan otomatis menggunakan localStorage.
